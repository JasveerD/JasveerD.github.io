01 Projects
BASIC to C Translator

A source-to-source compiler that translates a subset of BASIC into C. Implements a hand-written lexer, recursive descent parser, and AST-driven code emitter, built from scratch without parser-generator frameworks. A study in language structure, control flow translation, and the gulf between high-level syntax and low-level semantics.

C++ Compilers Parsing AST
Boid Simulation

A real-time 2D flocking simulation based on Craig Reynolds' boid algorithm, implemented in C++ with Raylib. Each agent applies separation, alignment, and cohesion rules locally and emergent collective behaviour arises from no global coordination. Built with a focus on spatial partitioning and frame-rate stability at high agent counts.

C++ Raylib Simulation Algorithms
Gravity Simulator

An n-body gravitational simulation implementing numerical integration of Newtonian dynamics across multiple particles. Explores the tradeoffs between integration accuracy and computational cost, with attention paid to floating-point precision and timestep stability.

C++ Numerical Methods Physics Simulation
02 Research

Graphene as a Material for Tritium and Deuterium Separation in Water

NSERC USRA · Pisana Lab, York University · May 2024 – Aug. 2024

Investigated graphene oxide membranes as a medium for isotopic water separation. Fabricated GO filters and prepared samples for structural and material characterization. Operated a Raman spectroscope to collect spectral data, estimating the degree of oxidation in GO membranes. Processed raw spectroscopy data in Python using SciPy for curve fitting and peak analysis. Presented findings at a summer research conference.

Python SciPy Raman Spectroscopy Materials Science
